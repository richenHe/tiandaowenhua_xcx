## 🔐 通用说明

### 接口调用方式

所有云函数接口统一通过以下方式调用：

```javascript
// 云函数调用方式
wx.cloud.callFunction({
  name: '云函数名',  // 如: user, order, course, ambassador, system
  data: {
    action: 'admin:接口名',  // 如: admin:getUserList
    ...其他参数
  }
})
```

### 统一响应格式

```json
{
  "success": true,          // 成功标识
  "code": 0,                // 状态码（0=成功，其他=错误）
  "message": "操作成功",     // 提示信息
  "data": {...}             // 返回数据
}
```

### 状态码说明

| 状态码 | 说明 |
|-------|------|
| 0 | 成功 |
| 400 | 参数错误 |
| 401 | 未授权（未登录或token过期） |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

### 鉴权说明

除登录接口外，所有管理端接口均需要在请求头中携带 JWT Token：

```javascript
// HTTP 请求头
headers: {
  'Authorization': 'Bearer {token}'
}
```

## 1. 认证与鉴权

### 1.1 管理员登录

**接口**: `system.admin:login`  
**说明**: 管理员登录，返回 JWT Token  
**鉴权**: 否

#### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| username | String | 是 | 用户名 |
| password | String | 是 | 密码（明文） |

#### 响应数据

```json
{
  "success": true,
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "admin": {
      "id": 1,
      "username": "admin",
      "real_name": "管理员",
      "role": "super_admin",
      "permissions": ["user:read", "order:write"]
    }
  }
}
```

#### 示例代码

```javascript
wx.cloud.callFunction({
  name: 'system',
  data: {
    action: 'admin:login',
    username: 'admin',
    password: '123456'
  }
}).then(res => {
  const { token, admin } = res.result.data;
  // 存储 token
  localStorage.setItem('admin_token', token);
});
```

### 4.2 创建课程

**接口**: `course.admin:createCourse`  
**说明**: 创建新课程  
**鉴权**: 是

#### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| name | String | 是 | 课程名称 |
| type | Number | 是 | 课程类型（1=初探班，2=密训班，3=咨询服务） |
| current_price | Number | 是 | 当前价格 |
| cover_image | String | 否 | 封面图片 |
| description | String | 否 | 课程简介（纯文本摘要，≤500 字）；传 `descriptionBlocks` 时由云函数根据文字块自动生成并覆盖 |
| descriptionBlocks | Array / JSON 字符串 | 否 | **2026-04-04** 简介图文块：`[{type:'text',text},{type:'image',image:cloudFileID}]` |
| content | String | 否 | 课程详情（长文）；**2026-04-04**：管理端「课程列表」新增/编辑弹窗已隐藏该字段，接口仍支持读写，编辑保存时会原样带回已有值 |
| outline | String | 否 | 课程大纲（JSON 字符串数组，仅文字）；传 `outlineBlocks` 时由服务端根据文字块重写 |
| outlineBlocks | Array / JSON 字符串 | 否 | **2026-04-04** 大纲图文块，格式同 `descriptionBlocks` |
| teacher | String | 否 | 讲师 |
| duration | String | 否 | 课程时长 |
| original_price | Number | 否 | 原价（默认等于current_price） |
| allow_retrain | Number | 否 | 是否允许复训（0=否，1=是） |

> **2026-04-04**：课程表已移除 `retrain_price`；复训价格在创建排期时通过 `createClassRecord.retrainPrice` 配置。
| sort_order | Number | 否 | 排序（默认0） |
| status | Number | 否 | 状态（0=下架，1=上架，默认1） |

#### 响应数据

```json
{
  "success": true,
  "message": "课程创建成功",
  "data": {
    "course_id": 16
  }
}
```

### 常用枚举值

#### 订单状态（pay_status）
- `0`: 待支付
- `1`: 已支付
- `2`: 已取消
- `3`: 已关闭
- `4`: 已退款

#### 课程类型（type）
- `1`: 初探班
- `2`: 密训班
- `3`: 咨询服务

#### 预约状态（status）
- `0`: 待确认
- `1`: 已确认
- `2`: 已签到
- `3`: 已取消

#### 大使等级（ambassador_level）
- `0`: 普通用户
- `1`: 初级大使
- `2`: 中级大使
- `3`: 高级大使
- `4`: 资深大使
- `5`: 特级大使

#### 申请状态（status）
- `0`: 待审核
- `1`: 已通过
- `2`: 已拒绝

#### 管理员角色（role）
- `super_admin`: 超级管理员
- `admin`: 管理员
- `operator`: 运营人员
